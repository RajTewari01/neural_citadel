# Experiment - PyQt Neural Citadel UI

This experiment tests the PyQt6 UI with:
- System stats monitoring (GPU/CPU/VRAM)
- Object detection with OpenCV + MediaPipe
- Looping video background
- Chat interface

## Setup
```bash
cd experiments/pyqt_neural_ui
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python main.py
```
